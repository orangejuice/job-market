Introduction to JSF

  • JavaServer Faces (JSF) is a framework appropriate for presentation-oriented applications
  • JSF is a component-based MVC framework built on top of Servlet API.
  • FacesServlet is sole request-response Controller, with Java Beans as Model and Facelets as View (JSP in older version)
  • JavaServer Faces provides:
    • API for representing components and managing their state
    • Handling events
    • Server-side validation and data conversion
    • Defining page navigation
    • Support for Â internationalization and accessibility
    • Extensibility for all these features
    • Tag libraries for adding components to web pages and for connecting components to server-side objects
