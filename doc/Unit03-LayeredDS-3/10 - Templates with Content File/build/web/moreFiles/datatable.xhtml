Data Tables

  • Tag that can represent Collections in tabular form
  • Only need to specify column information - automatically iterates through all elements to fill rows
  • Elements can be made active, e.g. to invoke methods such as sorting, editing, deleting.
  • Attribute value of dataTable indicates collection of data
  • Attribute var indicates name of individual collection elements
  • Basic structure:
      <h:dataTable value=\#{someBean.collection} var=ref>
        <h:column><f:facet name=“header”>title</facet>
          \#{ref.someattrib}</h:column>
        <h:column>...</h:column>
      </h:dataTable>
